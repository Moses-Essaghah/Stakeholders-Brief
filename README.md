# Stakeholders-Brief
This is an assignment on AI prompt engineering. A stakeholders brief was written for a retail company that has been experiencing a decline in performance for about 18 months. The entire brief was written by AI with a prompt that was developed using the RCTFCE framework.

The stakeholders brief was written in 4 different sections; The Situation Summary, Three Key Business Questions, Decisions These Questions Inform, and Description of Available Data.

After the Stakeholders brief, the exact AI prompt used was showen and the rationale for why my prompt was constructed the way it was was also given.
